Installation

To add this extension to your project, use the relevant command in your Quarkus project directory:

Quarkus CLI

quarkus ext add io.quarkus:quarkus-security-webauthn

Maven

./mvnw quarkus:add-extension -Dextensions="io.quarkus:quarkus-security-webauthn"

Gradle

./gradlew addExtension --extensions="io.quarkus:quarkus-security-webauthn"
