Installation

To add this extension to your project, use the relevant command in your Quarkus project directory:

Quarkus CLI

quarkus ext add org.apache.camel.quarkus:camel-quarkus-direct

Maven

./mvnw quarkus:add-extension -Dextensions="org.apache.camel.quarkus:camel-quarkus-direct"

Gradle

./gradlew addExtension --extensions="org.apache.camel.quarkus:camel-quarkus-direct"
